## 课程简介

本教程系统地介绍了 AI Agent 的核心概念、架构设计和实现方法。通过六个章节的学习，你将掌握：

- **基础概念**：理解 AI Agent 的定义、特征和演进历程
- **ReAct 模式**：掌握推理 + 行动的核心框架
- **工具系统**：学会创建和管理 Agent 工具
- **记忆系统**：实现短期和长期记忆管理
- **MCP 协议**：理解 Model Context Protocol 标准
- **Skills 系统**：为 Agent 添加专业化能力
