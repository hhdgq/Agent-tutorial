# AI Agent 教程

欢迎来到 AI Agent 教程！本教程将带你从零开始学习如何构建智能代理（AI Agent）。

---

## 教程概述

### 什么是 AI Agent？

AI Agent（智能代理）是指能够**感知环境**并通过**行动**来完成目标的智能系统。它结合了大型语言模型（LLM）的推理能力和工具系统的执行能力，可以自主完成复杂任务。

### 为什么学习 AI Agent？

- **未来趋势**：AI Agent 被认为是 AI 应用的下一个重要方向
- **实用技能**：掌握构建实际可用 AI 应用的能力
- **职业发展**：相关技能在就业市场的需求快速增长

## 课程结构

本教程包含六个核心章节：

### 第一章：AI Agent 基础概念
- 从图灵测试到现代 AI Agent
- Agent 的核心特征：自主性、反应性、主动性、社会性
- 完整架构解析：感知层、认知层、执行层
- LLM、Tools、Memory 三大核心组件

### 第二章：ReAct 模式详解
- ReAct（Reasoning + Acting）框架原理
- Thought-Action-Observation 循环
- 与其他推理方法的对比
- 从零实现 ReAct Agent

### 第三章：工具系统详解
- 工具的核心要素和设计原则
- JSON Schema 参数定义
- 创建自定义工具的完整指南
- 工具安全性考量
- 工具链和工作流

### 第四章：记忆系统详解
- 短期记忆与滑动窗口机制
- 长期记忆与向量数据库
- 语义搜索和相似度匹配
- 记忆压缩策略

### 第五章：MCP 协议详解
- Model Context Protocol 概述
- JSON-RPC 2.0 通信协议
- 三层架构：Host、Client、Server
- 与 Claude Desktop 集成

### 第六章：Skills 系统详解
- Skills 与 Tools 的区别
- Skill 核心架构
- SkillManager 技能管理器
- 创建自定义 Skills 实战

## 如何使用本教程

### 学习方式
1. **按顺序学习**：建议从第一章开始，逐步深入
2. **实践为主**：每章都有代码示例和练习
3. **循序渐进**：基础概念 → 核心框架 → 实战应用

### 学习提示
- 📖 **阅读理论**：理解核心概念和原理
- 💻 **运行代码**：亲手执行示例代码
- 🔧 **完成练习**：每章的实践练习
- 🤔 **思考问题**：章节末尾的思考题
